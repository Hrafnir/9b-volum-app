# 9B sin supre volum-app

En interaktiv web-app for ungdomsskolen som viser sammenhengen mellom måleenheter, prismevolum og liter.

Elevene kan:

- lage rektangulære prismer og terninger
- velge millimeter, centimeter, desimeter eller meter
- se volum i kubikkcentimeter, kubikkdesimeter og liter
- sammenligne liter med melkekartonger på 1 liter

Appen er statisk og kan åpnes direkte i nettleseren via `index.html`.
